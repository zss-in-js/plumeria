## Container functions

Argument: string  
The usage is the same as for media, and it can be used by putting it in an array.

```ts title="TypeScript"
import { css } from '@plumeria/core';

css.container.max();
css.container.min();
css.container.range();
```

- `css.container.max("width: 500px")`
- `css.container.min("width: 400px")`
- `css.container.range("width <= 400px")`
