It is the plumeria [Compiler `@plumeria/compiler` package](https://www.npmjs.com/package/@plumeria/compiler).  
Compiler uses [**rscute**](https://www.npmjs.com/package/rscute) is transpiled with [**@swc/core**](https://swc.rs/docs/usage/core) and executed in memory.
## Installation

<Tabs items={['npm',  'yarn', 'pnpm']}>
<Tab>
```sh title="Terminal"
npm i --save-dev @plumeria/compiler
```
</Tab>
<Tab>
```sh title="Terminal"
yarn add -D @plumeria/compiler
```
</Tab>
<Tab>
```sh title="Terminal"
pnpm i --save-dev @plumeria/compiler
```
</Tab>
</Tabs>
