It is the plumeria [vite-plugin `@plumeria/vite-plugin` package](https://www.npmjs.com/package/@plumeria/vite-plugin)  
Vite runs on the client, but Plumeria uses Node.js, so this is a plugin to prevent Node.js from being mixed into Vite.

## Installation
<Tabs items={['npm',  'yarn', 'pnpm']}>
<Tab>
```sh title="Terminal"
npm i -D @plumeria/vite-plugin
```
</Tab>
<Tab>
```sh title="Terminal"
pnpm i -D @plumeria/vite-plugin
```
</Tab>
<Tab>
```sh title="Terminal"
yarn add -D @plumeria/vite-plugin
```
</Tab>
</Tabs>
