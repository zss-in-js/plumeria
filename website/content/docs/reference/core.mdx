It is the plumeria [core `@plumeria/core` package](https://www.npmjs.com/package/@plumeria/core).  
This includes APIs exposed as css in js.
- `css.create()`
- `css.global()`
- `css.defineThemeVars()`
- `css.keyframes()`

## Installation

<Tabs items={['npm',  'yarn', 'pnpm']}>
<Tab>
```sh title="Terminal"

```
</Tab>
<Tab>

</Tab>
<Tab>
