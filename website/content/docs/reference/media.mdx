## Media functions

Argument: string | number
If it is a number, px will be added automatically.  
It can be put in an array and used as a selector `[css.media.maxWidth(560)]: {...}`.

```ts title="TypeScript"
import { css } from '@plumeria/core';

css.media.maxWidth();
css.media.maxHeight();
css.media.minWidth();
css.media.minHeight();
css.media.range();
```

- `css.media.maxWidth(764)`
- `css.media.maxWidth("48rem")`
- `css.media.range("width <= 800px")`
