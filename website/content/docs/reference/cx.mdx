The `cx()` function merges class names together while **filtering out falsy values** like `undefined`, `false`, or `null`. It is useful when conditionally combining styles.

```ts title="TypeScript"
import { css, cx } from '@plumeria/core';

const styles = css.create({
  one: {
    display: 'flex',
    justifyContent: 'space-between',
  },
  two: {
    fontSize: '14px',
    color: '#606060',
  },
});

const className = cx(styles.one, styles.two);  // [!code ++] â†’  "one_ocpc7u two_ocpc7u"
```

## Usage Patterns

### Conditional styles

```ts
const isActive = true;

const className = cx(
  styles.one,
  isActive && styles.two // 'styles.two' is only added when isActive is true
);
```

### With ternary

```ts
const className = cx(
  isDark ? styles.dark : styles.light,
  isHovered && styles.hovered
);
```

---

## Return value
Returns a single string of space-separated class names, ignoring falsy inputs:

```ts
cx('a', false, undefined, 'b'); // â†’ 'a b'
```
