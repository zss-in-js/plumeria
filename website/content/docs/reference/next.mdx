It is the plumeria [Server Component `@plumeria/next` package](https://www.npmjs.com/package/@plumeria/next)  
ServerCSS is rendered using `router.refresh()`.

## Installation
<Tabs items={['npm',  'yarn', 'pnpm']}>
<Tab>
```sh title="Terminal"
npm i --save-dev @plumeria/next
```
</Tab>
<Tab>
```sh title="Terminal"
yarn add -D @plumeria/next
```
</Tab>
<Tab>
```sh title="Terminal"
pnpm i --save-dev @plumeria/next
```
</Tab>
</Tabs>

## Overview
```ts
export { ServerCSS } from './component/ServerCSS';
```

```jsx
import { ServerCSS } from '@plumeria/next';
```
