## Installation

<Tabs items={['npm',  'yarn', 'pnpm']}>
<Tab>
```sh title="Terminal"
npm i --save-dev eslint-plugin-zss-lint
```
</Tab>
<Tab>
```sh title="Terminal"
pnpm i --save-dev eslint-plugin-zss-lint
```
</Tab>
<Tab>
```sh title="Terminal"
yarn add -D eslint-plugin-zss-lint
```
</Tab>
</Tabs>
