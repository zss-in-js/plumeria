It is the plumeria [`@plumeria/webpack-plugin` package](https://www.npmjs.com/package/@plumeria/webpack-plugin)  
AST parse compiles plumeria styles into CSS (HMR)

## Overview
```ts title="TypeScript"
import type { Compiler } from 'webpack';
export declare class PlumeriaPlugin {
    private outputFileName;
    private stylesByFile;
    private outFile;
    constructor(outputFileName?: string);
    apply(compiler: Compiler): void;
    registerFileStyles(filePath: string, styles: Partial<any>): void;
    private generateOrderedCSS;
    private writeCSS;
}
```
