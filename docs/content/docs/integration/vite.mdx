[`@plumeria/vite`](https://www.npmjs.com/package/@plumeria/vite)  is the Vite plugin for Plumeria.  
It compiles styles into CSS via AST parsing, with HMR support.

## Installation
<Tabs items={['npm',  'yarn', 'pnpm']}>
<Tab>
```sh title="Terminal"
npm i -D @plumeria/vite-plugin
```
</Tab>
<Tab>
```sh title="Terminal"
yarn add -D @plumeria/vite-plugin
```
</Tab>
<Tab>
```sh title="Terminal"
pnpm i -D @plumeria/vite-plugin
```
</Tab>
</Tabs>

## Overview
```ts title="TypeScript"
import type { Plugin } from 'vite';
export function plumeria(): Plugin;
```

## API
```ts title="vite.config.ts"
import { plumeria } from '@plumeria/vite';

export default defineConfig({
  plugins: [plumeria()],
});
```
