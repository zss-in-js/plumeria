`createTheme` for defining CSS variables for each theme.

## Example

```ts
import * as css from '@plumeria/core'

export const theme = css.createTheme({
  colors: {
    default: 'black',
    dark: 'white'
  },
  bg: {
    default: 'white',
    dark: 'black'
  }
})
```

This will generate the following CSS

```css
:root {
  --colors: black;
  --bg: white;
}

[data-theme="dark"] {
  --colors: white;
  --bg: black;
}
```

### Using Themes

To use themes, you need to add the `data-theme` attribute to the root element.

```tsx
<html data-theme="dark">
  <body>
    ...
  </body>
</html>
```



## Media Queries

`createTheme` can also be used with media queries.

```ts
export const theme = css.createTheme({
  colors: {
    default: 'black',
    dark: 'white',
    '@media (prefers-color-scheme: dark)': 'white'
  }
})
```

This will generate the following CSS

```css
:root {
  --colors: black;
}

[data-theme="dark"] {
  --colors: white;
}

@media (prefers-color-scheme: dark) {
  :root {
    --colors: white;
  }
}
```


<Callout title="Good to know">
All APIs, including createTheme, are not compiled until the variable is used.
</Callout >
