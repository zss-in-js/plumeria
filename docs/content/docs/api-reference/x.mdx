title: x
description: 'API reference for the x function'
---

The `x` function allows you to pass dynamic values (like state or props) into your static styles via CSS variables.

## Example

Pass your `css.props()` result as the first argument, and an object of CSS variables as the second.

```tsx
import { useState } from 'react';
import { css, x } from '@plumeria/core';

const styles = css.create({
  bar: {
    width: 'var(--width)',
    background: 'aqua',
  },
});

export function Component() {
  const [width, setWidth] = useState(0);
  
  return (
    <div {...x(css.props(styles.bar), { '--width': width + 'px' })} />
  );
}
```

The `x` function returns an object with `className` and `style` properties that can be spread directly onto an element.

<Callout title="Use sparingly">
  `x()` is powerful, but it comes with a small runtime cost due to inline styles. For static styles, prefer using `css.create()` and `css.props()` wherever possible.
</Callout>
