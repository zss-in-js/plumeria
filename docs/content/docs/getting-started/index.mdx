## Install Plumeria



<Tabs items={["npm", "yarn", "pnpm"]}>
  <Tab value="npm">
  ```bash title="Terminal"
  npm install -D @plumeria/core @plumeria/compiler
  ```

  </Tab>
  <Tab value="yarn">
  ```bash title="Terminal"
  yarn add -D @plumeria/core @pumeria/compiler
  ```

  </Tab>
  <Tab value="pnpm">
  ```bash title="Terminal"
  pnpm add -D @plumeria/core @plumeira/compiler
  ```

  </Tab>
</Tabs>

To learn more about installing `@plumeria`, see the [installation guide](/docs/getting-started/installation).
