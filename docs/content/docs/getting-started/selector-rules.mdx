## Nesting
Pseudo-elements(class) and media nesting rules. The following example is acceptable:

```ts title="TypeScript" {1}
import { css } from '@plumeria/core';

const styles = css.create({
  list: {
    '@media (max-width: 768px)': { // [!code ++]
      '& li': {
        listStyleType: 'none',
      },
    },  // [!code ++]
  },
});
```



<Callout type="error" title="error cases">
  Unacceptable examples:
</Callout>

```ts title="TypeScript" {1}
import { css } from '@plumeria/core';

const styles = css.create({
  list: {
    '& li': {
      '@media (max-width: 768px)': { // [!code --]
        listStyleType: 'none',
      }, // [!code --]
    },
  },
});
```

```ts title="TypeScript" {1}
import { css } from '@plumeria/core';

const styles = css.create({
  list: {
    ':hover': {
      '@media (max-width: 768px)': { // [!code --]
        background: 'skyblue',
      }, // [!code --]
    },
  },
});
```


<Callout type="info" title="Nesting Restrictions">
  Media can only be nested once with pseudo and &, but pseudo and & cannot nest media.
</Callout>
