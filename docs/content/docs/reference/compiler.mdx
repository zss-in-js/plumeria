It is the plumeria [Compiler `@plumeria/compiler` package](https://www.npmjs.com/package/@plumeria/compiler).  
A Rust-based compiler with static extraction in the style of Plumeria.
## Installation

<Tabs items={['npm',  'yarn', 'pnpm']}>
<Tab>
```sh title="Terminal"
npm i --save-dev @plumeria/compiler
```
</Tab>
<Tab>
```sh title="Terminal"
yarn add -D @plumeria/compiler
```
</Tab>
<Tab>
```sh title="Terminal"
pnpm i --save-dev @plumeria/compiler
```
</Tab>
</Tabs>
