It is the plumeria [Server Component `@plumeria/next` package](https://www.npmjs.com/package/@plumeria/next)  
ServerCSS is rendered using `router.refresh()`.

## Installation
<Tabs items={['npm',  'yarn', 'pnpm']}>
<Tab>
```sh title="Terminal"
npm i --save-dev @plumeria/next-plugin
```
</Tab>
<Tab>
```sh title="Terminal"
yarn add -D @plumeria/next-plugin
```
</Tab>
<Tab>
```sh title="Terminal"
pnpm i --save-dev @plumeria/next-plugin
```
</Tab>
</Tabs>

## Overview
```ts title="TypeScript"
import type { NextConfig } from 'next';
export declare function withPlumeria(nextConfig?: NextConfig): NextConfig;
```

## API
```ts title="next.config.ts"
import type { NextConfig } from "next";
import { withPlumeria } from "@plumeria/next-plugin";

const nextConfig: NextConfig = withPlumeria({
  /* config options here */
});

export default nextConfig;
```
