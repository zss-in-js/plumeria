It is the plumeria [vite-plugin `@plumeria/vite` package](https://www.npmjs.com/package/@plumeria/vite)  
Vite runs on the client, but Plumeria uses Node.js, so this is a plugin to prevent Node.js from being mixed into Vite.

## Installation
<Tabs items={['npm',  'yarn', 'pnpm']}>
<Tab>
```sh title="Terminal"
npm i -D @plumeria/vite
```
</Tab>
<Tab>
```sh title="Terminal"
yarn add -D @plumeria/vite
```
</Tab>
<Tab>
```sh title="Terminal"
pnpm i -D @plumeria/vite
```
</Tab>
</Tabs>

## Overview
```ts
import type { Plugin } from 'vite';
export default function plumeria(): Plugin;
```

## API
```ts
import plumeria from '@plumeria/vite';

export default defineConfig({
  plugins: [plumeria()],
});
```
