import { Tab, Tabs } from 'fumadocs-ui/components/tabs';

The `global` method defines styles that apply globally to the entire document.  
It supports HTML elements, class names, and pseudo-classes or media queries â€” all **with full type support** and **compile-time extraction**.

<Callout title="Semmantic API">
Keep global styles to a minimum to reduce bundle size.
</Callout>


## Example

<Tabs items={['ts', 'css']}>
  <Tab>

```ts title="TypeScript"
import { css } from '@plumeria/core'

css.global({
  ":root": {
    '--main-color': '#333',
  },
  "*": {
    boxSizing: "border-box"
  },
  html: {
    margin: 0,
    padding: 0,
  },
  body: {
    margin: '0 auto',
    color: 'white',
    background: '#303030',
  },
  h2: {
    fontSize: '22px',
    fontWeight: '550',
    color: 'white',
  },
});
```
</Tab>
