## Pseudo items

string | Functions argument: string  
Just like media and container, you can use it as a selector by putting it in an array.  
You can write pseudo of multiple using cx.

```ts title="TypeScript"
import { css } from '@plumeria/core';

css.pseudo.active;
css.pseudo.hover;
css.pseudo.focus;
css.pseudo.link;
css.pseudo.visited;
css.pseudo.target;

css.pseudo.nthLastChild();

css.pseudo.nthOfType();
```
