## Return Value

Filters undefined and false and returns a string.

```ts
const styles = css.create({
  one: {
    display: 'flex',
    justifyContent: 'space-between',
  },
  two: {
    fontSize: '14px',
    color: '#606060',
  },
});
```

```ts title="tsx"
import { cx } from '@plumeria/core';

const style = cx(styles.one, styles.two);
```
